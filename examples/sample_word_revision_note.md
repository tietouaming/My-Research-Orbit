# Word And MathType Revision Note

Problem: A manuscript edit request includes prose, equation references, captions, and
MathType formula objects.

Action: Create a backup, inspect the selection, enable tracked revisions, and use comments
for uncertain edits. Do not treat formula objects as normal text.

Rule: If MathType is unavailable, report explicit failure instead of replacing an equation
with plain text.

Verified: The workflow is a planning template and performs no document write in dry-run.
