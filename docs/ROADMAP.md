# Roadmap

## Version 0.1

- Python package and CLI.
- Dry-run workflow execution.
- Memory-card generator.
- Redaction audit.
- Provider registry.
- Local private application-pack generation.
- Local web workbench for project inspection.
- Tests and GitHub Actions CI.

## Version 0.2

- Better memory-card extraction from longer mixed notes.
- More workflow fixtures and expected-output comparisons.
- Richer redaction categories with allowlist support.
- Optional JSON reports for all CLI commands.
- UI controls for launching dry-run workflows from selected examples.

## Version 0.3

- Tested direct Claude and Gemini transports.
- Structured COMSOL summary import helpers.
- More detailed Word/MathType planning reports.
- Multi-machine memory merge reports.

## Long Term

- Research operations dashboard with execution history.
- Pluggable evaluators for workflow quality.
- Provider comparison harness using desensitized fixtures.
- Reproducible research-agent benchmark tasks.
