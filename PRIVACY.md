# 隐私说明

Research Orbit 采用本地优先的隐私边界设计。

## 公开仓库内容

公开仓库只应包含源代码、workflow 定义、测试、文档、小型脱敏示例和公开示例输出。

## 本地私有内容

以下内容必须只保留在本地：

- 生成的申请材料；
- 私有操作日志；
- 真实 API key 或 provider 凭据；
- 原始 COMSOL 模型和仿真输出；
- 本机绝对路径、账号相关元数据；
- 未公开文档或未脱敏研究细节。

## 外部 provider

默认 `dry-run` provider 不会向外部服务发送数据。只有用户显式配置 provider 环境变量，并主动运行相关命令时，才可能发生真实模型请求。
